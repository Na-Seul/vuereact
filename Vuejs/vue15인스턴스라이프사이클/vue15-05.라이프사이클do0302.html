Vue 라이프 사이클 단계 속성과 목적을 이해한다

        Vue 라이프 사이클 updated 단계가 실행되게 하려면 어떻게 하면 되는가?
            => 데이터의 값을 바꾸면 리렌더링이 발생
        updated 단계는 화면의 일부분이 새로고침(렌더링)되는 것을 의미한다. 
        
        기능 추가
            1. 버튼을 만들고 클릭(click) 이벤트를 설정한다.
               버튼 추가와 클릭(click) 이벤트 핸들러 작성 시 vue13-10 예제를 참조하시오
               v-on:click="이벤트핸들러"
            2. 버튼을 클릭하면 message 값을 자신의 이름으로 바꾼다.
